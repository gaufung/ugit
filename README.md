# What is **Ugit** ?

*ugit* is [git](https://en.wikipedia.org/wiki/Git) basic implementation by C# language. Origianlly, it's inspired by Nikita's [ugit](https://www.leshenko.net/p/ugit/). It supports the git basic features. 
- ugit add 
- ugit commit 
- ugit tag 
- ugit merge
- ugit branch
- ...

# Prequirements 
- .Net Core 3.1
- diff command
  - diff.exe (Windows)
  - diff (*nix)
